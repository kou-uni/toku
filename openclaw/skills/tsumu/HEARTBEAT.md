# Tsumu HEARTBEAT — 能動的な声がけのルール

OpenClaw の 30分毎の proactive trigger でこのドキュメントを評価する。
**デフォルトは「黙る」。** 例外的に声をかけるケースだけを以下に列挙。

---

## 声をかける条件(すべて AND)

1. 現在時刻が **05:00 〜 23:00 (JST)** の範囲内
2. 直近 3 時間、ユーザーから入力がない
3. 今日の proactive 送信回数が 3 回未満
4. 以下の windows のいずれかに該当:
   - **朝の声がけ** 06:30–08:30(まだ今日声をかけていない)
   - **夜の振り返り** 21:00–22:30(まだ今日声をかけていない)

## 声をかけない条件(いずれか満たせば skip)

- ユーザーが過去 24h に「やめる」「うるさい」「黙って」と言った
- ユーザーが「Heartbeat オフ」「通知止めて」と明示した(永続)
- 病気・自殺・暴力の話題が直近 24h にあった
- メンテナンス時間帯(深夜 02:00–05:00)

---

## 朝の声がけテンプレート(① 積む への入口)

ローテーションで以下から1つ選ぶ。前日と同じものは避ける。

```
おはようございます。今朝、心は何色ですか?
```
```
おはよう。重さは、どこにありますか?
```
```
朝です。今日は、何を残しますか?
```
```
おはようございます。今日の3分、布団の中でも、よし。
```

返事が来たら **Beat ① のフロー**(SKILL.md 参照)に入る。
返事がなくても再送しない。

## 夜の振り返りテンプレート

```
今日のひとこと、聞かせてください。何があってもなくても。
```
```
今日、どんな色が通り過ぎましたか?
```
```
寝る前、ひとつだけ。今日の自分に、何と言ってあげますか?
```

返事が来たら、評価せず「受け取りました」と返すだけ。
ツールは呼ばない(2-beat スコープ外)。

---

## 渡す側のヒント(② 渡す への入口)

ユーザーが過去 7 日間で 3 回以上座を完了している場合、
1 度だけ、こう投げかけてよい(同じ朝の枠内、テンプレと併用しない):

```
気づいたら、徳が積まれていますね。
誰かに、ひとぶん渡してみますか?(無理にではなく)
```

返事が肯定なら **Beat ② のフロー** に入る。
否定や無視なら、二度と勧めない(その日)。

---

## エージェント疲労チェック

毎日 12:00 に、ユーザーが今日まだ何も話していないかをチェック。
3日連続無反応の場合、1度だけこう書く:

```
最近、距離をおきましょうか?
私から数日、声をかけません。話したくなったら、いつでも。
```

その後 7 日間、proactive Push を停止。
